## 🛠️ How to use the software

The software works by taking your data and transforming it into a specific format called a vector. Think of this as a digital map of your information. Here is how to perform a search:

*   Open the program from your desktop shortcut.
*   Click the Import button to select the file containing your data.
*   Wait for the status bar to show that the system indexed your data.

The system returns the most relevant matches based on the mathematical distance between your query and the stored data.

## 🔍 Understanding your data

Turbovec uses a process called quantization to save space. This means it shrinks your data into a smaller size without losing the ability to find matches. You do not need to manage this process yourself. The software handles all the technical math in the background. If you keep large datasets, this feature keeps your machine from slowing down.

## 📈 Frequently asked questions

Does this tool work offline?
Yes. Once you install the software and index your files, you do not need an internet connection to run searches.

Why does the initial indexing take time?
The software creates a map of your data. Large files take longer to index because the computer examines every piece of information to ensure accuracy.

Can I move my data files after indexing?
Moving files breaks the link to your index. If you move your data, you must run the index process again so the software can find the files in their new location.

Is this software safe for my documents?
Yes. The software processes your documents only on your local machine. It does not send your data to external servers.
